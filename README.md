The Event Management Portal is a simple and user-friendly web application developed using HTML, CSS, and JavaScript to help users manage events efficiently. It allows users to add, edit, delete, search, filter, and sort events while displaying key details such as date, time, venue, organizer, category, and participant capacity. The portal also features a dashboard with event statistics, status badges, countdown timers, dark mode, and local storage to save data, providing a clean, responsive, and efficient event management experience.
